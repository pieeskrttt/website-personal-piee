Pembelajaran tentang Ilmu Web

HTML

HTML adalah bahasa markup untuk membuat struktur halaman web. Mulai dari heading, paragraf, gambar, hingga tautan.

CSS

CSS digunakan untuk mengatur tampilan halaman web, seperti warna, layout, animasi, dan responsivitas.

JavaScript

JavaScript adalah bahasa pemrograman yang menambah interaktivitas pada website, seperti animasi, validasi form, dan konten dinamis.

Node.js

Node.js adalah runtime JavaScript di server yang memungkinkan membuat aplikasi backend dan API yang cepat dan scalable.

Bootstrap

Bootstrap adalah framework CSS populer untuk membangun website responsif dengan komponen siap pakai.

MySQL

MySQL adalah sistem manajemen basis data relasional yang digunakan untuk menyimpan dan mengelola data aplikasi.

Tailwind CSS

Tailwind CSS adalah framework utility-first yang memudahkan styling dengan kelas-kelas siap pakai secara cepat.
